## Issues Found & Resolutions

### Issue 1: AsyncError Test Timeout
**Problem:** Test "mirrors AsyncError when assetProvider errors" failed with:
```
Expected: <Instance of 'AsyncError'>
Actual: AsyncLoading<Portfolio>:<AsyncLoading<Portfolio>()>
```

**Root Cause:** portfolioProvider was still in AsyncLoading state when checked. Error hadn't propagated yet due to timing.

**Resolution:** Explicitly trigger assetProvider.future and catch the exception before reading portfolioProvider state. This forces synchronous error propagation.

**Code Change:**
```dart
// Before: Simple delay
await Future<void>.delayed(const Duration(milliseconds: 100));

// After: Explicit error trigger
try {
  await container.read(assetProvider.future);
} catch (e) {
  // Error expected
}
```
