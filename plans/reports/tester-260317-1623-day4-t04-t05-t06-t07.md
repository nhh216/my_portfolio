## Issues Encountered & Resolved

### Issue 1: Extension Methods Lost in Dynamic Cast
**Severity:** HIGH | **Status:** RESOLVED

**Problem:**
```dart
// Original code that failed:
final asset = assets.cast<dynamic>().firstWhere(
  (a) => a.id == assetId,
  orElse: () => null,
);
// Error: Class '_$AssetImpl' has no instance getter 'totalValue'
```

**Root Cause:** Casting to `dynamic` stripped type information, making extension methods inaccessible.

**Fix Applied:**
```dart
// Corrected code:
Asset? asset;
try {
  asset = assets.firstWhere((a) => a.id == assetId);
} catch (_) {
  asset = null;
}
```

**Files Modified:**
- `/Users/hung/Desktop/Project/portfolio_tracker/lib/features/portfolio/presentation/screens/asset_detail_screen.dart`
